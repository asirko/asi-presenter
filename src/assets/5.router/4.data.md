### Les datas

Il est possible de passer des données de paramétrage dans les routes :

```typescript
const routes: Routes = [
  {
    path: '',
    component: LayoutComponent,
    data: { val: 3 },
  },
];

@NgModule({
  imports: [RouterModule.forRoot(routes)],
  exports: [RouterModule],
})
export class CoreRoutingModule {}
```
